Proyecto: Sistema de Restaurante

Objetivo:

Desarrollar una aplicación o sistema que permita simular el manejo de clientes y solicitudes para un restaurante de comida rápida haciendo uso de Estructuras de Datos cómo las Pilas y Colas para dar funcionalidad al sistema.

Alcance:

Crear un sistema que permita manejar el sistema de pedidos y el sistema de la fila de clientes esperando. El sistema de pedidos debe poder administrar la creación de cada pedido y añadir las posibilidades de completar y eliminar dichos pedidos, en base a este sistema se cambiará el sistema de clientes que muestra la fila esperando a realizar su pedido.

Funcionalidades:

  • Registro de pedidos

    El sistema debe mostrar un registro de cada pedido siguiendo una serie de pasos interactivos que llenen una orden final.

  • Mostrar pedidos

    El sistema debe tener un apartado donde muestre todos los pedidos, máximo 4 activos para no saturar la cocina.

  • Organizar pedidos

    El sistema debe mostrar los pedidos activos y los pedidos ya completados.

  • Eliminar pedidos

    En la vista de pedidos, se debe tener la opción de cancelar o eliminar pedidos.

  • Manejar extras en el pedido

    El sistema debe permitir al cliente hacer solicitudes extra a sus pedidos que incrementer su orden y por ende el costo de la misma.
